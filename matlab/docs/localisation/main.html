function main(eval, trials, varargin)

MAIN repository for evaluation trials

This function provides several different evaluation trials.

Example: To run an evaluation trial with for , one could enter the following command:

>> main('T60', 50)

Arguments

  • eval (str): name of evaluation trial to run (required)
  • trials (int): number of trials to run (default: 100)
  • varargin (cell array): variable additional arguments (i.e. used to provide non-default max. em-iteration range for em-iteration trial)
clearvars('-except', 'eval', 'trials', 'varargin');
if nargin < 2, trials=100; end
